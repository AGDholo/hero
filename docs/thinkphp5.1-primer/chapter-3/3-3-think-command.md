:::note 提示 

Think 命令是 ThinkPHP 出品的一款控制台程序，包括不限于进行 生成模块、创建类库、清除缓存、生成缓存、创建模型等操作。 

:::

以后的章节所有能够使用命令行的都不再手动创建文件。  
详细文档请参考: [Think 命令官方文档](https://www.kancloud.cn/manual/thinkphp5_1/354138)
