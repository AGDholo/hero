## 建立目的

在业务开发中有大量的重复功能需要开发者去编写，并且代码质量很难可控。  
为了解决这些问题，Hero Admin 采用插件式的方法，依赖 Laravel IOC 容器进行自动注入，
实现完全解耦的业务代码复现。
